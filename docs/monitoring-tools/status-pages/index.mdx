# Status Pages

Light weighted status pages softwares

## Cachet

- **Language/Stack**: PHP (Laravel)
- **Highlights**:

  - Cachet itself is strictly a status page and incident‐reporting system.
  - Built-in monitoring: No. Feed to Cachet’s API
  - Beautiful, responsive dashboard
  - Incident reporting & scheduled maintenance notifications
  - Metrics graphs (via Grafana integration or built-in)
  - RSS, Atom & JSON API feeds for easy integration
- **Repo**: [https://github.com/CachetHQ/Cachet](https://github.com/CachetHQ/Cachet)

## Uptime Kuma

- **Language/Stack**: Node.js (with Electron desktop app)
- **Highlights**:

  - Uptime Kuma is both monitor and status page in one package
  - Simple, modern UI with real-time updates
  - Supports HTTP(s), TCP, ping, DNS, Docker container, and more checks
  - Push notifications: Telegram, Discord, Slack, email, SMS
  - Built-in status page & customizable notifications
- **Repo**: [https://github.com/louislam/uptime-kuma](https://github.com/louislam/uptime-kuma)
